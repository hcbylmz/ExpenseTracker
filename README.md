# Expense Tracker MVP


## Tech Stack

- **Expo + React Native + TypeScript**
- **SQLite** (expo-sqlite) - Data storage
- **i18next** - Multi-language support (EN/TR)
- **Async Storage** - Theme and currency preferences
- **react-native-chart-kit** - Data visualization
- **expo-file-system + expo-sharing** - CSV export

## Features

Expense tracking | Budget management | Visual analytics | Recurring expenses | Income tracking | Multi-currency | Dark/Light theme | Multi-language | CSV export

AI-powered development tools truly boost productivity!
